## :white_check_mark:&nbsp;주요 기능
- Log in, Log out, Sign up 기능
- 실시간 채팅 기능
- 메세지 작성 시간 화면에 표시 기능
- 특정 사용자가 작성한 메세지 선별 기능
- 자신이 작성한 메세지 삭제 기능
- 사용자가 프로필 사진을 업로드하지 않았을 경우, 이름의 첫글자로 표시 기능
